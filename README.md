# CS_522_HW3

#### About the project:
The main aim of this project is to simulate an experiment for the Fitt's Law.

<b>Fitt's Law:</b> (ID) = log(D/W +1)


ID: the index of difficulty.<br/>
D: the distance from the starting point to the center of the target.<br/>
W: the width of the target measured along the axis of motion.<br/>

#### How to Run: 
Please download and open the sim.html in any browser.
